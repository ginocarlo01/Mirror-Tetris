Para visualizar este exercício, é necessário acessar localhost/phpmyadmin, criar um banco de dados com o nome "myDB".

Além disso, é necessário ter o XAMPP configurado e inicializado previamente.

Após isso, coloque os arquivos enviados na pasta htdocs da instalação do XAMPP.

Acesse então o arquivo createTables.php, responsável por inicializar a tabela.

Depois disso, você pode chamar o arquivo SignUpPage.html e utilizar o site.
